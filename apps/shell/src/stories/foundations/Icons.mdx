# Icons

O projeto usa exclusivamente **Material Icons**. Nunca use SVGs customizados ou outras bibliotecas.

## Como usar

```tsx
<span className="material-icons text-icon-sm text-primary-900">
  menu
</span>
```

## Tamanhos

| Token | Size | Tailwind |
|---|---|---|
| `text-icon-xs` | 20px | `text-icon-xs` |
| `text-icon-sm` | 24px | `text-icon-sm` |
| `text-icon-md` | 32px | `text-icon-md` |
