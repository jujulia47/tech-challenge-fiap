# Breakpoints & Layout

## Sistema de breakpoints

| Name | Min width | Layout | Nav |
|---|---|---|---|
| Mobile (default) | 0px | Single column | `MenuDrawer` overlay |
| `md` tablet | 768px | Single column | `MenuTabs` horizontal |
| `lg` desktop | 1024px | Sidebar + main | `MenuSidebar` vertical |
