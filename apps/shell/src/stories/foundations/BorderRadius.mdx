## Border tokens

| Token | Value | Tailwind | Usage |
|---|---|---|---|
| `border-width-1` | 1px | `border` | Input/dropdown stroke |
| `border-color-default` | `color-primary-900` | `border-primary-900` | Input/dropdown closed |
| `border-color-focus` | `color-primary-800` | `border-primary-800` | Input focus / Select open state |
| `border-color-error` | `color-error` | `border-error` | Error state |

## Elevation

O design não usa box shadows. Elevação é transmitida exclusivamente através de contraste de cor de fundo.
